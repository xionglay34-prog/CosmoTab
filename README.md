# CosmoTab

把标签页变成宇宙银河 —— 一键归类、去重、按访问时间排序。

## 安装

```bash
npm install
npm run build
```

构建产物会输出到 `dist/`。

## 加载到 Chrome

1. 打开 `chrome://extensions`。
2. 右上角开启「开发者模式」。
3. 点击「加载已解压的扩展程序」，选择本仓库的 `dist` 目录。
4. 在工具栏点击 CosmoTab 图标即可使用。

## 功能

- **一键整理**：自动去重 + 按域名分组（tabGroups）+ 按最近访问排序。
- **去重**：严格（URL 完全相同）或宽松（origin + pathname 相同）。
- **排序**：按 `lastAccessed` 时间倒序排列；支持窗口内排序或合并到一个窗口。
- **宇宙视图（深空模式）**：每个窗口是一个星系，每个域名是一颗行星，每个标签页是绕行星运行的卫星，最近访问的卫星更亮、更靠近行星。
- **明亮模式**：以卡片形式展示，按窗口和域名分组。

## 隐私

仅使用 `tabs`、`tabGroups`、`storage`、`favicon` 权限，无网络请求，所有数据存储在本地。
