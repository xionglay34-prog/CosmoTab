### 详细介绍（16,000 字内 · 详情页主体）

```
🌌 CosmoTab — 把混乱的标签页变成宇宙银河

如果你和我一样，一开 Chrome 就有 30+ 个标签页飘在窗口上方变成像素，CosmoTab 就是为你设计的。

✨ 核心功能

· 一键整理 ：把所有打开的标签按域名分组，归入 Chrome 原生标签组
· 智能去重 ：三种模式可选 —
  - Strict：URL 完全相同才算重复
  - Loose：去 hash、追踪参数、参数排序后相同
  - Domain：同一域名下只保留最近访问的一个
· 最近访问排序 ：自动把刚用过的标签置顶，按 1m / 13m / 2h 显示相对时间
· 撤销关闭 ：误关了？一键 Undo（基于 Chrome Sessions API）
· 白名单保护 ：固定的 / 加白的 tab 永不会被自动关闭
· 侧边栏视图 ：在浏览器侧栏快速浏览所有标签，无需切窗口

🔒 隐私至上

CosmoTab 完全在你本地运行：
· 不收集任何数据
· 不发送任何网络请求
· 所有偏好（白名单、主题、模式）只保存在 chrome.storage.local
· 开源代码可审计，欢迎来仓库 issue / PR

🎨 设计

· 浅色 / 深色双主题
· 手绘风插画与微动效，告别冷冰冰的工具感
· 中文友好的相对时间表达（刚刚 / 5m / 2h / 3d）

🚀 上手

1. 安装后点击工具栏 CosmoTab 图标
2. 侧边栏会自动打开
3. 点 "一键整理" 即开始

💌 反馈与建议

GitHub: https://github.com/Leonxlnx/CosmoTab
有想法、找 bug、想加功能，欢迎来 issue 区。

---

更新日志见仓库 CHANGELOG。
```
