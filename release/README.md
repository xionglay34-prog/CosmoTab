## CosmoTab · Chrome Web Store 发布材料包

本目录包含上架 Chrome Web Store 所需的全部素材与文档。  
按 `01-checklist.md` 一步步走即可。

### 目录索引

| 文件 | 用途 |
| --- | --- |
| `01-checklist.md` | **总流程清单** — 注册账号到上线的逐步操作 |
| `02-store-listing.md` | **商品详情文案** — name / 简短说明 / 详细介绍 / 类别 / 关键词 |
| `03-permissions.md` | **权限解释** — 每个 permission 的用途说明（必填） |
| `04-privacy-policy.md` | **隐私政策正文** — 拷贝到 GitHub Pages 即得公开 URL |
| `05-screenshots.md` | **截图脚本** — 怎么用 DevTools 拍 1280×800 + 哪些页面要拍 |
| `06-faq.md` | **审核被拒常见问题 + 答复模板** |
| `dist.zip` | 构建产物（运行 `npm run release` 自动生成） |
| `screenshots/` | promo 图（运行 `npm run promo` 自动生成） |
